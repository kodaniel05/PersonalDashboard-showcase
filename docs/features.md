# Features

## Dashboard Navigation

The dashboard is organized into focused views: Home, Productivity, School, Personal, Relationships, Finance, and Explore. Each view groups widgets around a real workflow instead of showing every tool at once.

## Daily Snapshot

The Home view provides daily context such as weather, holiday information, headlines, moon phase, time-zone awareness, and music context when configured. It is designed to work even when optional services are unavailable.

## Planning and Productivity

Planning features include today’s plan, recurring plans, important dates, term practice, and document organization. The goal is to keep deadlines, routines, and supporting materials close to the daily workflow.

## Document Vault

The document vault organizes uploaded or linked documents by category, with search and filtering behavior. Public screenshots should use fake document names and non-sensitive files.

## Finance Workspace

Finance tools include savings visualization, paycheck prediction, weekly history, and a market watchlist. The design supports live, delayed, and offline states for market data.

Personal widgets support reflection, good moments, bucket lists, playlists, career timeline tracking, and small memory-focused records.

## Travel and Food Exploration

Explore features include a travel map and restaurant/food adventure tracking, allowing places, trips, and ideas to be collected visually.

## Customization and Resilience

The dashboard supports themes, persisted active views, collapsible widgets, responsive layouts, and error boundaries. External integrations use clear fallback states when credentials are missing or services are unavailable.
